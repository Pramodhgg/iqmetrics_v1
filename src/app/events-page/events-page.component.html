Computer Vision, Machine and Deep Learning

Duration: 4 Weeks

Mode : Online

Rating : (18)

Objective

Objective of this course to help developers, researchers, and students just like yourself become experts in machine learning and deep learning. We will be teaching the Python programming language for all the examples. Tensorflow, OpenCV, Keras, Scikit-Learn, Numpy, Pandas, Matplot and etc. We will use Goolge Colab platform for hands on training of deep learning models.

Outcome of the course

  • He/She will be able to build computer vision applications using OpenCV and Tensorflow.

    • Face Recognition

    • Age Estimation

    • Facial Expression Recognition

  • He/She will be able to Natural Language Processing using NLTK and Tensorflow.

    • Sentiment Classification

    • SMS Spam Classification

  • He/She will be able to build machine learning and deep learning models for image and text related dataset.

  • He/She will deploy the AI model in Mobile.

  • He/She will deploy the AI application in IOT devices like Raspberry Pi or Jetson Nano

Course Curriculum

  • 1) What is Machine Learning

  • 2) Crash Course of Python Programming (10 hours)

  • 3) Getting Started with AI environment setup

  • 4) Computer vision and image processing basics

    • Loading, Displaying and saving Images.

    • Drawing Operations.

    • Basic Image Processing

    • Kernals

    • Morphological Operations

    • Smoothing & blurring

    • Thresholding

    • Gradient &Edge Detection

    • Contours

    • Histograms

  • 5) Image descriptors

    • Color Channel Statistics

    • Color Histograms

    • Haralick Texture

    • Local Binary Patterns

    • HOG

    • Key Point detection

  • 6) He/She will deploy the AI application in IOT devices like Raspberry Pi or Jetson Nano
